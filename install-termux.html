📱 Termux 建站超详细教程

一、Termux 下载与安装

  1. 不要直接在酷安/应用商店下载安装！
    推荐到 F-Droid 或 GitHub官方Releases 下载最新版 Termux APK。
  2. 安装后第一次打开时，可能会弹出存储权限提示，建议同意。
  3. 如果看到 Unable to resolve host 等错误，检查网络和软件源。

二、更新软件源与环境

  1. 更换国内源（提升下载速度）：
    输入 termux-change-repo，选择国内的源（如清华、中科大等），按空格选中后回车，再按回车确认。
  2. 更新包管理器和升级环境：
    pkg update && pkg upgrade -y
git：pkg install git -y
  • 安装 curl：pkg install curl -y
  • 安装 nano/vim（文本编辑器）：pkg install nano -y
  • 安装 openssh（用于 SSH 连接）：pkg install openssh -y
  • 安装 Python（可选）：pkg install python -y
  • 安装 Node.js（可选）：pkg install nodejs -y

    • 四、配置 SSH 密钥用于 GitHub 推送

    1. 生成 SSH 密钥：
      ssh-keygen -t ed25519 -C "你的邮箱"
      一路回车即可。
    2. 查看公钥内容：
      cat ~/.ssh/id_ed25519.pub
      长按终端复制输出的全部内容。
    3. 添加公钥到 GitHub：
      打开 GitHub SSH Keys 页面，点击 New SSH key，把刚才复制的内容粘贴进去并保存。
    4. 测试是否成功：
      ssh -T git@github.com
      如果显示 Hi 用户名! You've successfully authenticated 就说明配置成功。

    五、克隆 GitHub 仓库到本地

    1. 找到你要操作的 GitHub 仓库，点击 Code，选择 SSH，复制仓库地址（如 git@github.com:Alhkxsj/Alhkxsj.github.io.git）。
    2. 在 Termux 运行：
      git clone 仓库地址
      例如：
      git clone git@github.com:Alhkxsj/Alhkxsj.github.io.git
    3. 进入仓库目录：
      cd Alhkxsj.github.io

    六、编辑和上传你的网页

    1. 用 nano 编辑网页文件：
      nano install-termux.html
      编辑后，Ctrl+O 保存，Ctrl+X 退出。
    2. 查看修改状态：
      git status
    3. 提交修改并推送到 GitHub：
      git add . git commit -m "更新说明" git push

    七、常见问题与解决办法

    • Q: Termux 软件源很慢/下载失败
      A: 用 termux-change-repo 更换国内源。
    • Q: git push 时提示权限错误
      A: 检查 SSH 密钥是否添加到 GitHub，并且用的是 SSH 地址而不是 HTTPS。
    • Q: 编辑文件时乱码
      A: 建议用 nano，存为 UTF-8 编码。
