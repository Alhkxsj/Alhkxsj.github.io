🔧 Termux环境配置指南

一、Termux简介

Termux是一款强大的Android终端模拟器，让你在手机上像Linux一样操作命令行，适合学习Linux、Python、网页开发等。

二、Termux安装

  1. 推荐方式：前往 F-Droid 或 GitHub官方 下载最新版Termux。
  2. 安装完成后，首次启动会自动配置基础环境。

三、基础环境配置

  1. 更新软件源： 
    pkg update && pkg upgrade
  2. 安装常用工具： 
    pkg install git vim python nano curl wget
  3. 设置中文支持（可选）： 
    pkg install zh-CN
export LANG=zh_CN.UTF-8

四、SSH密钥生成（用于GitHub操作）

  1. 生成密钥： 
    ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
  2. 查看公钥并复制到GitHub： 
    cat ~/.ssh/id_rsa.pub
    登录GitHub → Settings → SSH and GPG keys → New SSH key，粘贴公钥内容。

五、常用命令总结

  • ls：查看当前目录文件
  • cd 目录名：进入目录
  • pwd：显示当前路径
  • cp 源文件 目标文件：复制文件
  • mv 源文件 目标文件：移动/重命名文件
  • rm 文件名：删除文件
  • nano 文件名：编辑文件（nano为简易编辑器）

六、常见问题与解决方法

  • Q: 安装或更新时报错怎么办？
    A: 先执行 pkg update，如仍有问题请检查网络或更换镜像源。
  • Q: 无法访问外网？
    A: 检查手机代理设置，或尝试切换网络环境。
  • Q: 如何恢复Termux默认设置？
    A: 删除 ~/.termux 目录或卸载重装应用。
