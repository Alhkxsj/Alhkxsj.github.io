🔧 Termux建站环境配置详细指南

一、Termux简介

Termux 是一款强大的 Android 终端模拟器和 Linux 环境应用，它能让你在手机或平板上直接使用 Linux 命令行工具。

主要特点：

  • 无需 root 权限，即可体验完整 Linux 终端环境
  • 支持包管理，随时安装 git、vim、python 等开发工具
  • 可配合 GitHub 等平台，实现代码托管、静态网站部署
  • 可通过 SSH、SFTP 等方式远程管理服务器
  • 丰富的第三方插件（API、Widget、Styling等），扩展性极强
应用场景：
  • 随时随地在手机上编辑、管理和同步网站源码
  • 学习 Linux 基础命令与开发流程
  • 作为轻量服务器终端，远程管理 VPS、云服务器等
  • 辅助搭建个人博客、静态网站等

二、下载安装与初始配置

  1. 下载安装 Termux：
    • 推荐从 F-Droid 或 GitHub发布页 获取最新版 APK。
    • 注意：不要在 Google Play 或第三方应用市场下载，避免版本过旧或被植入广告。
  2. 安装并首次启动：
    • 安装 APK 后，点击图标打开 Termux。
    • 首次启动会自动初始化基础环境，无需额外操作。
  3. 授予存储权限：
    • 在 Termux 中输入以下命令（首次需要）：
    termux-setup-storage
    • 系统会弹出权限申请对话框，请选择“允许”。
    • 完成后，Termux 主目录会出现 storage 文件夹，方便访问/管理手机存储中的文件。

三、更新系统与安装基础开发工具

  1. 更新包管理器与系统：
    • 输入以下命令，确保软件源和系统为最新：
    pkg update && pkg upgrade -y
    • 如遇网络慢，可参考后文更换国内镜像源。
  2. 安装常用开发工具：
    • 一条命令安装建站必需工具：
    pkg install git vim nano python curl wget openssh -y
    • git：同步代码，连接 GitHub
    • vim/nano：编辑站点源码和配置文件，新手推荐 nano
    • python：部分静态网站生成器或脚本必需
    • curl/wget：下载工具
    • openssh：生成 SSH 密钥，远程管理服务器
  3. （可选）安装中文支持，防止乱码： 
    pkg install language-pack-zh-cn
export LANG=zh_CN.UTF-8
    • 如需永久生效，可将 export LANG=zh_CN.UTF-8 写入 ~/.bashrc 或 ~/.profile。

四、配置 Git 与 SSH 密钥（推送代码到 GitHub 必备）

  1. 配置 Git 用户信息： 
    
git config --global user.name "你的昵称"
git config --global user.email "你的邮箱@example.com"
    • 用于标记你的提交身份。
  2. 生成 SSH 密钥对： 
    ssh-keygen -t rsa -b 4096 -C "你的邮箱@example.com"
    • 一路按回车即可，密钥默认保存于 ~/.ssh/ 目录。
  3. 查看并复制公钥内容： 
    cat ~/.ssh/id_rsa.pub
    • 复制输出内容
    • 登录 GitHub，依次点击 Settings > SSH and GPG keys > New SSH key，粘贴公钥，保存。
  4. 测试 SSH 连接： 
    ssh -T git@github.com
    • 首次连接会询问是否信任，输入 yes 并回车。
    • 若显示 “Hi 用户名! You've successfully authenticated...” 即配置成功。
