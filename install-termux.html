一、Termux 下载与安装

  1. 不要直接在酷安/应用商店下载安装！
    推荐到 F-Droid 或 GitHub Releases 获取最新版。
  2. 安装后第一次打开时，建议同意存储权限。
  3. 如果看到 Unable to resolve host 等错误，检查网络源与环境。

二、更换国内源与更新环境

  1. 更换国内源（提升下载速度）：
    输入 termux-change-repo，选择国内的源（如清华、中科大），按空格选中、回车确认。
  2. 更新包管理器和升级环境：
    pkg update && pkg upgrade -y

三、安装常用关键工具

  1. 安装 git：pkg install git -y
  2. 安装 curl：pkg install curl -y
  3. 安装 nano/vim（编辑器）：pkg install nano -y
  4. 安装 openssh（SSH 工具）：pkg install openssh -y
  5. 生成密钥（一路回车即可）：
    ssh-keygen -t ed25519 -C "你的邮箱"
  6. 查看公钥内容，长按终端复制输出的全部内容：
    cat ~/.ssh/id_ed25519.pub
  7. 添加公钥到 GitHub：
    打开 GitHub SSH Keys，点击 New SSH key，粘贴内容保存。
  8. 测试 SSH 连接：
    ssh -T git@github.com
    显示 Hi 用户名! You've successfully authenticated 即成功。

四、克隆 GitHub 仓库到本地

  1. 找到你的 GitHub 仓库，复制 SSH 地址（如 git@github.com:Alhkxsj/Alhkxsj.github.io.git）。
  2. 在 Termux 运行：
    git clone 仓库地址
    例如：git clone git@github.com:Alhkxsj/Alhkxsj.github.io.git
  3. 进入仓库目录：
    cd Alhkxsj.github.io

五、编辑和上传你的网站

  1. 编辑网页文件（例如）：
    nano index.html
    编辑后 Ctrl+O 保存，Ctrl+X 退出。
  2. 查看修改状态：
    git status
  3. 提交并推送到 GitHub：
    git add .
git commit -m "更新说明"
git push

六、常见问题与解决办法

  • Q: Termux 软件源很慢/下载失败
    A: 用 termux-change-repo 更换国内源。
