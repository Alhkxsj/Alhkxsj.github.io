## 项目简介

本项目致力于为零基础用户详细讲解如何使用 **Termux**（Android 终端模拟器）和 **GitHub Pages** 免费搭建个人或项目静态网站。教程内容涵盖环境配置、网页上传、域名绑定、内容创作等各个环节，适合所有希望用移动设备实现建站的小伙伴。

---

## 项目特色

- 🧭 **超详细分步教程**：从 Termux 安装、环境配置到网站部署，每一步配有命令说明，图文并茂。
- 📱 **移动端友好体验**：页面自适应设计，手机也能流畅访问和操作。
- ✨ **美观的页面设计**：玻璃拟态 UI、粒子背景、炫酷导航栏滑块。
- 🔒 **安全高效**：介绍 SSH 密钥推送、常见报错解决等进阶技巧。
- 🌍 **开源共享**：教程和代码完全开源，欢迎学习](https://alhkxsj.github.io/)

## 快速开始

### 1. 克隆本仓库

```bash
git clone https://github.com/Alhkxsj/Alhkxsj.github.io.git
```

### 2. 自定义你的网站内容

- 修改 `index.html`、`about.html` 等页面内容
- 替换你的图片、教程、样式等资源

### 3. 上传到你的 GitHub Pages 仓库

```bash
git add .
git commit -m "Customize my site"
git push
```

### 4. 绑定自定义域名（可选）

详见 [项目部署](github-pages.html) 页面第五部分。

## 常见问题

- **Q: 为什么网站打不开？**  
  A: 请确认仓库名、index.html 位置是否正确，推送后请耐心等待几分钟。

- **Q: 如何用手机推送网页？**  
  A: 参考 [环境配置](install-termux.html) 的 SSH 配置与 git 推送章节。

- **Q: 可以自定义主题和内容吗？**  
  A: 当然可以！只需修改 HTML/CSS 文件即可。

- **Q: 支持移动端直接操作吗？**  
  A: 支持，教程全部适配手机端，Termux 即可完成全部操作流程。
