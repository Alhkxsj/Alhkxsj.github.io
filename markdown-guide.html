✍️ Markdown高效写作

一、Markdown简介

Markdown是一种轻量级标记语言，简洁易学，非常适合写文档、博客、项目说明等。

二、基础语法

  • 标题：
    # 一级标题
## 二级标题
### 三级标题
  • 加粗/斜体：
    **加粗**
*斜体*
  • 无序/有序列表：
    - 无序列表项
- 另一项

1. 有序列表项
2. 另一项
  • 插入链接和图片：
    [链接文字](https://example.com)
![图片描述](图片地址)
  • 代码：
    `行内代码`

```python
print("代码块")
```

三、进阶用法

  • 表格：
    | 姓名 | 年龄 |
| ---- | ---- |
| 张三 | 20   |
| 李四 | 22   |
  • 引用：
    > 这是引用内容
  • 任务列表：
    - [x] 已完成
- [ ] 未完成

四、Markdown预览工具

  • GitHub、Gitee等平台自带预览。
  • 本地推荐 Typora、Mark Text、VS Code 插件。

五、常见问题

  • Q: 图片无法显示？
    A: 路径要用相对或绝对链接，注意文件名大小写，图片要已提交到仓库。
  • Q: 语法无效？
    A: 检查符号是否英文状态，缩进和空格是否正确。
  • Q: 如何插入目录？
    A: 手动添加锚点，或使用 VS Code 插件自动生成。
