一、Markdown 简介

Markdown 是一种轻量级文档标记语言，广泛用于博客、项目文档、技术写作等，兼容性极强，写作高效。

二、基础语法一览

  • 标题： 
    # 一级标题
## 二级标题
### 三级标题
  • 加粗/斜体/删除线： 
    **加粗文本**
*斜体文本*
~~删除线~~
  • 引用： 
    > 这是引用内容
  • 无序/有序列表： 
    - 项目A
- 项目B

1. 第一项
2. 第二项
  • 链接与图片： 
    [百度](https://www.baidu.com)
![图片Alt](https://github.com/favicon.ico)
  • 代码与代码块： 
    `行内代码`

```bash
echo Hello Markdown
```

三、进阶用法

  • 任务列表： 
    - [x] 已完成
- [ ] 未完成
  • 表格： 
    | 姓名 | 年龄 |
| ---- | ---- |
| 张三 | 18   |
| 李四 | 22   |
  • 多级列表： 
    - 一级
  - 二级
    - 三级
  • 注释： 
    <!-- 这是注释 -->
  • 脚注： 
    Markdown 很棒[^1]。
[^1]: 这里是脚注内容。

四、常用工具推荐

  • 网页版：markdown.com.cn/editor
  • 本地软件：Typora、Mark Text、Obsidian
  • VS Code 插件：Markdown All in One、Markdown Preview Enhanced
  • 平台自带：GitHub、Gitee、语雀等

五、常见问题

  • Q: 图片无法显示？
    A: 检查图片路径、文件名大小写、图片已上传。
  • Q: 换行无效？
    A: 行末加两个空格，或用 <br> 强制换行。
  • Q: 如何插入目录？
    A: 可用 [TOC] 或工具自动生成。
  • Q: .md 文件能直接作为网页吗？
    A: GitHub Pages 支持 Jekyll，可自动渲染 Markdown。
