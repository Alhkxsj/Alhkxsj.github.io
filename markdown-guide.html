一、Markdown 简介

Markdown 是一种轻量级标记语言，帮助你用极简语法快速写出结构清晰、格式优雅的文档、博客、作品集。

二、常用 Markdown 语法速查

  • 标题：# 一级标题，## 二级标题，以此类推
  • 粗体：**加粗**，斜体：*斜体*
  • 无序列表：- 项目1，有序列表：1. 项目1
  • 插入链接：[链接文本](https://...)
  • 插入图片：![图片说明](图片地址)
  • 代码块：`行内代码`，多行用```语言包裹
  • 引用：> 引用内容
  • 分割线：--- 或 ***

三、示例：写一篇 Markdown 博客

# 我的第一篇博客

## 简介
大家好，这是我的第一篇 Markdown 博客！

四、Markdown 编辑工具推荐

  • VS Code（手机/电脑均可用，配合 Markdown Preview 插件）
  • Typora（极致简洁，所见即所得，适合桌面端）
  • StackEdit（网页版，无需安装，点此访问）
  • 手机推荐 JotterPad、Markor、MWeb、飞书文档等

五、将 Markdown 转换为网页

  1. 用 Typora 或 StackEdit 导出为 HTML 文件。
  2. 将导出的 HTML 文件上传到你的 GitHub Pages 仓库。

六、进阶玩法

  • 自定义 CSS 美化导出的 HTML 页面。
  • 用 Hexo、Hugo 等静态博客框架批量管理 Markdown 博客。
  • 在 GitHub Pages 上自动部署 Markdown 博客。
