✍️ Markdown 写作指南

一、Markdown 简介

Markdown 是一种轻量级标记语言，帮助你用极简语法快速写出结构清晰、格式优雅的文档、博客、作品集。

二、常用 Markdown 语法速查

  • 标题：# 一级标题，## 二级标题，以此类推
  • 粗体：**加粗**，斜体：*斜体*
  • 无序列表：- 项目1，有序列表：1. 项目1
  • 插入链接：[链接文本](https://...)
  • 插入图片：![图片说明](图片地址)
  • 代码块：`行内代码`，多行用```语言包裹
  • 引用：> 引用内容
  • 分割线：--- 或 ***

三、示例：写一篇 Markdown 博客

# 我的第一篇博客

## 简介
大家好，这是我的第一篇 Markdown 博客！
