✍️ Markdown 高效写作

一、Markdown 简介

Markdown 是一种轻量级标记语言，广泛用于文档、博客、代码说明等场景。其语法简洁明了，便于书写和阅读。

二、基础语法

  • 标题： 
    # 一级标题
## 二级标题
### 三级标题
  • 段落与换行： 
    这是一个段落。
这是同一个段落，
    这是换行。

    段落之间空一行，换行可用两个空格或 <br> 标签。

  • 加粗、斜体、删除线： 
    **加粗文本**
*斜体文本*
~~删除线~~
  • 引用： 
    > 这是引用内容
  • 无序/有序列表： 
    - 无序列表项1
- 无序列表项2

1. 有序列表项1
2. 有序列表项2
  • 链接与图片： 
    [百度](https://www.baidu.com)
![Logo](https://github.com/favicon.ico)
  • 分割线： 
    ---
  • 行内代码和代码块： 
    `行内代码`

```python
print("Hello, Markdown!")
```

三、进阶用法

  • 任务列表： 
    - [x] 已完成
- [ ] 未完成
  • 表格： 
    | 姓名 | 年龄 |
| ---- | ---- |
| 张三 | 20   |
| 李四 | 22   |
  • 多级列表： 
    - 一级
  - 二级
    - 三级
  • 注释： 
    <!-- 这是注释，不会在页面显示 -->
  • 脚注： 
    Markdown 是一个很棒的工具[^1]。
[^1]: 这里是脚注内容。

四、常用编辑器与预览工具

  • 在线编辑器：markdown.com.cn/editor
  • 本地软件：Typora、Mark Text
  • VS Code 插件：Markdown All in One
  • 平台自带：GitHub、Gitee、语雀等都支持 Markdown 预览

五、常见问题与排查

  • Q：图片无法显示？
    A：检查图片路径是否正确，文件名大小写是否一致，图片已上传到仓库。
  • Q：语法无效？
    A：符号需为英文状态，缩进空格要规范。
  • Q：如何插入目录？
    A：手动添加锚点或使用 VS Code 插件自动生成目录。
  • Q：换行失效？
    A：末尾加两个空格或使用 <br> 实现换行。
